# ATTENDX

ATTENDX is an Android-based attendance management application designed for educational institutions. The app enables teachers to mark attendance digitally, manage student records, and monitor attendance efficiently.

## Features
- Teacher login authentication
- Student attendance tracking
- Attendance percentage calculation
- Firebase integration
- User-friendly Android UI

## Technologies Used
- Java
- Android Studio
- Firebase
- XML

## Project Duration
Dec 2024 – Mar 2025
